# Daily Tasks Web App

A simple and clean *Daily Task Manager* built using *HTML, CSS, and JavaScript*.
This web application helps users organize their daily activities by adding, tracking, and completing tasks.

## 🚀 Live Demo

🔗 https://maitrishivhare.github.io/dailytask-app/

## 📂 GitHub Repository

🔗 https://github.com/maitrishivhare/dailytask-app

## ✨ Features

* Add new tasks easily
* View pending tasks
* Mark tasks as completed
* Clean and responsive user interface
* Simple task management for daily productivity

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📸 Project Preview

Daily Tasks app allows users to manage their day-to-day tasks efficiently with a minimal and modern design.
